Problems that Responsive Web Design meant to solve

Responsive Web design is a technique meant to help make websites be compatible to mobile devices as well as to traditional computer screens and practically to any screen resolution. Since it is impractical to design custom websites for varying screen resolution, screen orientation and for any device out there, people realized that it is essential to come up with a technology that would allow us to design websites that respond to user's screen resolution and platform. Hence the Responsive Web Design approach to web design was created to make websites adapt an adjustable screen resolution as well as to repond automatically to landscape and portrait orientation of devices at any instance.

The concept of Responsive web design was introduced by Ethan Marcotte and he wanted the website to be flexible like a room that adjusts to the number of occupants within it. With Responsive Web Design, flexibile design of websites can be employed to make websites fit automatically into any screen resolution.
